## Decisions Made

- CSS variables on document.documentElement for global dark mode without React prop drilling or CSS-in-JS
- Theme persisted in localStorage; falls back to prefers-color-scheme media query on first visit
- Pomodoro uses useRef for intervalRef to avoid stale closures in setInterval callback
- PR Queue fetch triggered by githubData useEffect so it only runs when GitHub is actually connected
- Task sort applied after pin toggle locally (not full refetch) to preserve pinned-first order instantly
- labels guarded with `(pr.labels || [])` to handle missing field from API gracefully
