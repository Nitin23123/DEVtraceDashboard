<task type="auto">
  <name>Task 1: Create tasks controller</name>
  <files>backend/src/controllers/tasksController.js</files>
  <action>
Create `backend/src/controllers/tasksController.js` with five exported async handler functions. All handlers import `{ query }` from `'../db'` and use `req.user.id` for user scoping.

**getTasks(req, res):**
```sql
SELECT * FROM tasks WHERE user_id = $1 ORDER BY created_at DESC
```
Returns `200 { tasks: rows }`.

**createTask(req, res):**
Destructure `{ title, description, priority, due_date }` from `req.body`. Validate `title` is present — return `400 { error: 'Title is required' }` if missing. Insert:
```sql
INSERT INTO tasks (user_id, title, description, priority, due_date)
VALUES ($1, $2, $3, $4, $5)
RETURNING *
```
Default status is 'todo' (DB default). Returns `201 { task: rows[0] }`.

**getTask(req, res):**
```sql
SELECT * FROM tasks WHERE id = $1 AND user_id = $2
```
Returns `404 { error: 'Task not found' }` if no row. Returns `200 { task: rows[0] }`.

**updateTask(req, res):**
Destructure `{ title, description, status, priority, due_date }` from `req.body`. Build dynamic SET clause — only update provided fields plus always set `updated_at = NOW()`. Use:
```sql
UPDATE tasks SET title = COALESCE($1, title), description = COALESCE($2, description),
  status = COALESCE($3, status), priority = COALESCE($4, priority),
  due_date = COALESCE($5, due_date), updated_at = NOW()
WHERE id = $6 AND user_id = $7
RETURNING *
```
Returns `404` if no row updated. Returns `200 { task: rows[0] }`.

**deleteTask(req, res):**
```sql
DELETE FROM tasks WHERE id = $1 AND user_id = $2 RETURNING id
```
Returns `404` if no row deleted. Returns `200 { message: 'Task deleted' }`.

Wrap each handler body in try/catch — on error call `next(err)` or return `500 { error: 'Server error' }`.

Export all five: `module.exports = { getTasks, createTask, getTask, updateTask, deleteTask }`.

<task type="auto">
  <name>Task 2: Create tasks router and mount in app.js</name>

  <action>
Create `backend/src/routes/tasks.js`:
```javascript
const express = require('express');
const router = express.Router();
const { verifyToken } = require('../middleware/auth');
const { getTasks, createTask, getTask, updateTask, deleteTask } = require('../controllers/tasksController');

router.get('/',     verifyToken, getTasks);
router.post('/',    verifyToken, createTask);
router.get('/:id',  verifyToken, getTask);
router.put('/:id',  verifyToken, updateTask);
router.delete('/:id', verifyToken, deleteTask);

module.exports = router;
```

Then open `backend/src/app.js` and add the tasks router mount after the auth router line:
```javascript
const tasksRouter = require('./routes/tasks');
// ...
app.use('/api/tasks', tasksRouter);
```

The mount must come before the 404 fallthrough handler. Do not change any existing auth routes, middleware, or error handlers.
