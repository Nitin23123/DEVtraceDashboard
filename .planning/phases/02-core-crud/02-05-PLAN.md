<task type="auto">
  <name>Task 1: Create goals controller</name>
  <files>backend/src/controllers/goalsController.js</files>
  <action>
Create `backend/src/controllers/goalsController.js` with five exported async handler functions. All handlers import `{ query }` from `'../db'` and use `req.user.id` for user scoping.

**getGoals(req, res):**
```sql
SELECT * FROM goals WHERE user_id = $1 ORDER BY created_at DESC
```
Returns `200 { goals: rows }`.

**createGoal(req, res):**
Destructure `{ title, description, target_date }` from `req.body`. Validate `title` is present — return `400 { error: 'Title is required' }` if missing. Insert:
```sql
INSERT INTO goals (user_id, title, description, target_date)
VALUES ($1, $2, $3, $4)
RETURNING *
```
`is_completed` defaults to `false` (DB default). Returns `201 { goal: rows[0] }`.

**updateGoal(req, res):**
Destructure `{ title, description, target_date }` from `req.body`. Use COALESCE:
```sql
UPDATE goals SET title = COALESCE($1, title), description = COALESCE($2, description),
  target_date = COALESCE($3, target_date), updated_at = NOW()
WHERE id = $4 AND user_id = $5
RETURNING *
```
Returns `404` if no row updated. Returns `200 { goal: rows[0] }`.

**completeGoal(req, res):**
This is the dedicated complete action — it only flips `is_completed`:
```sql
UPDATE goals SET is_completed = true, updated_at = NOW()
WHERE id = $1 AND user_id = $2
RETURNING *
```
Returns `404 { error: 'Goal not found' }` if no row. Returns `200 { goal: rows[0] }`.

**deleteGoal(req, res):**
```sql
DELETE FROM goals WHERE id = $1 AND user_id = $2 RETURNING id
```
Returns `404` if no row deleted. Returns `200 { message: 'Goal deleted' }`.

Wrap each handler body in try/catch — on error return `500 { error: 'Server error' }`.

Export all five: `module.exports = { getGoals, createGoal, updateGoal, deleteGoal, completeGoal }`.

<task type="auto">
  <name>Task 2: Create goals router and mount in app.js</name>
  <files>backend/src/routes/goals.js, backend/src/app.js</files>
  <action>
Create `backend/src/routes/goals.js`. Note the complete endpoint uses `PUT /:id/complete` — this must be declared BEFORE the generic `PUT /:id` so Express matches it correctly:
```javascript
const express = require('express');
const router = express.Router();
const { verifyToken } = require('../middleware/auth');
const { getGoals, createGoal, updateGoal, deleteGoal, completeGoal } = require('../controllers/goalsController');

router.get('/',           verifyToken, getGoals);
router.post('/',          verifyToken, createGoal);
router.put('/:id/complete', verifyToken, completeGoal);  // MUST come before /:id
router.put('/:id',        verifyToken, updateGoal);
router.delete('/:id',     verifyToken, deleteGoal);

module.exports = router;
```

Then open `backend/src/app.js` and add the goals router mount:
```javascript
const goalsRouter = require('./routes/goals');
// ...
app.use('/api/goals', goalsRouter);
```

Mount before the 404 fallthrough handler. Do not disturb existing routes or middleware.
